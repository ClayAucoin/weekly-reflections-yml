This week we got into changing content on pages and submitting forms with JavaScript.

One thing I had an issue with and had to do some research on was having my page immediately refresh with my form submit posting to the same page. I found that when using a button with the type as submit, I needed to add the code below to my function to prevent the refresh back to the forms initial state.

    event.preventDefault();
